How to Split a PDF
Split Options:
  • Split into individual pages: Creates separate PDF files for each page
  • Extract specific pages: Extract only the pages you specify
Page Range Examples:
  • 1-5 - Extract pages 1 through 5
  • 3 - Extract only page 3
  • 1,3,5-7 - Extract pages 1, 3, and 5 through 7
Note: If multiple files are created, they will be packaged in a ZIP file for download.
